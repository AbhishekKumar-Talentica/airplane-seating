# airplane-seating
Airplane seating system

## <u>Input</u>
seat = {{1,2},{3,4}};

passenger = 12;

## <u>Output</u>
<pre>
___________________________________________________________________________
|  [] : Represent one seat                                                |
|  <> : Represent Seat type Aisle,window,Middle                           |
|  PA : Represent passenger number                                        |
___________________________________________________________________________
</pre>
